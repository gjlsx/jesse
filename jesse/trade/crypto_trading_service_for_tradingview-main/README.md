1.稍微小改一下TradingView的交易策略代码，这一部分很简单，人人都能学会，我的Youtube频道和B站频道会教大家如何修改  
2.我们需要一台海外服务器来运行这个软件，不建议使用个人电脑，因为不稳定，而且一般没有外网IP。具体如何部署请看我的另一个视频和文章 如何在服务器上运行这个软件
